Постановка задачи:
Моделирование и исследование библиотечной системы с использованием графовых возможностей SQL Server  


Цель:
Разработать и исследовать модель библиотечной системы, включающую книги, авторов, жанры и читателей, с использованием графовых возможностей SQL Server  


Описание задачи:
Необходимо создать графовую базу данных в SQL Server, которая моделирует связи между книгами, авторами, жанрами и читателями, а также их взаимодействия (выдача, возврат, предпочтения).
