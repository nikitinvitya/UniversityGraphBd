Постановка задачи:
Моделирование и исследование социальных связей студентов с использованием графовых возможностей SQL Server Management Studio


Цель:
Разработать и исследовать модель социальных связей студентов, их обучения в университетах и проживания в городах с использованием графовых возможностей SQL Server Management Studio  


Описание задачи: 
Создание графовой базы данных в SQL Server Management Studio, которая моделирует связи между студентами, их дружеские отношения, места обучения и проживания.
